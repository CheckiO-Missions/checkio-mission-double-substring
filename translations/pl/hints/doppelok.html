Przydatne będzie przeczytanie kilku informacji na temat: str.count() (example), list.append() (example), len() (example), max() (example) i range() (example).

Proszę o pomoc!

Zacznijmy od zastanowienia się, jak utworzyć różne podciągi z elementów, które znajdują się obok siebie. W tym celu użyjemy dwóch pętli for i dodatkowej zmiennej num.

line = "aaaa"
num = 1
for a in range(len(line)):  # In this cycle, we determine how much the length
            # of the substring will change in one cycle on the given line
    for i in range(len(line)):  # In this cycle, we define the substrings themselves
        substring = line[i:i+num]  # A substring is formed by this expression
        print(substring)
    num += 1  # This variable will increase by 1 after each cycle
            # and it will determine the length of the substring

Co robić dalej?

Następnie proszę pominąć to wszystko w warunku if. Interesują nas podciągi z więcej niż jednym w rzędzie. Proszę użyć str.count() jako warunku.

line = "aaaa"
num = 1
for a in range(len(line)):
    for i in range(len(line)):
        if line.count(line[i:i+num])> 1:
            substring = line[i:i+num]
            print(substring)
    num += 1

Utknąłem. Potrzebuję małej podpowiedzi.

Nie pozostało już wiele rzeczy do zrobienia. Musi Pan teraz utworzyć listę, na której można zapisać długości podciągów, które występują więcej niż raz w rzędzie. W tym celu proszę użyć metody list.append().

num = 1
result = []
for a in range(len(line)):
    for i in range(len(line)):
        if line.count(line[i:i+num])> 1:
            result.append(len(line[i:i+num]))
    num += 1
Jak zwrócić poprawną odpowiedź?

Proszę zawinąć wszystko w wyrażenie funkcyjne i użyć funkcji max(). 0 zostało ustawione jako domyślny argument, więc jeśli ciąg jest pusty, nasza funkcja zwróci 0.

def double_substring(line: str) -> int:

    num = 1
    result = []
    for a in range(len(line)):
        for i in range(len(line)):
            if line.count(line[i:i+num])> 1:
                result.append(len(line[i:i+num]))
        num += 1
    return max(result, default=0)

Dobra, dobra! To jest duch!=) To rozwiązanie będzie trudne do zrozumienia. Moduł "re" jest tutaj używany. Za pomocą wyrażenia "re.findall(r'(?=(.*).*\1)', line)" i funkcji len() z łatwością uzyskamy pożądany wynik.

import re


def double_substring(line: str) -> int:

    return max(map(len, re.findall(r'(?=(.*).*\1)', line)))
Chcę więcej!

Proszę spojrzeć na to rozwiązanie. Zbudowane na pętli while i sprawdzaniu podciągów. Logika jest nieco podobna do naszego rozwiązania, pętla while zakończy się, gdy podciąg będzie miał długość całego ciągu. Funkcja max() jest również używana do zwrócenia wyniku.

def double_substring(line: str) -> int:

    res = 0
    x = 1
    while x <= len(line):
        if line[:x] in line[x:]:
            res = max(res,len(line[:x]))
            x += 1
        else:
            line = line[1:]
    return res
